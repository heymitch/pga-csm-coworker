# Status Reports Playbook

## BOD Report

**When:** Every weekday morning before standup
**Where:** #ghostwriting-success-ops

### First Month Template
```
BOD Date [MM/DD/YYYY]

Active Clients: XX
EEC Launch Status:
Completed Assignment 0: XX
Completed Assignment 1: XX
Completed Assignment 2: XX
Completed Assignment 3: XX
```

### Standard Template (Month 2+)
```
BOD Date [MM/DD/YYYY]

Active Students: XX
EEC Launches This Week: XX
Paid Clients This Month: XX

Assignment Pipeline:
- Module 0-1: XX students
- Module 2-3: XX students (EEC building)
- Module 4-5: XX students (outreach prep)
- Module 6-7: XX students (active outreach/sales)
- Module 8-9: XX students (fulfillment)

Flags:
- [Student] inactive X days
- [Student] failed payment, escalation stage X
- [Student] unsigned agreement, X days outstanding

Projections:
- [Student] likely to launch EEC this week (submitted Assignment 3 yesterday)
- [Student] Liftoff conversation targeted for [date]
```

### Data Sources
- AirTable: Student View (stages, dates, KPIs)
- Slack: Recent assignment submissions, activity
- SamCart: Payment status
- AirTable "All Contracts Pending" view

## EOD Report

**When:** Every weekday evening
**Where:** #ghostwriting-success-eod

### Template
```
EOD [MM/DD/YYYY]

Calls: [X] completed ([list: Student - Call Type])
Assignments Reviewed: [X] ([list: Student - Module])
Questions Answered: [X]
Check-ins Sent: [X]
Failed Payments Contacted: [X]

Notable:
- [Key observation or student win]
- [Any escalation or flag]

Tomorrow Priority:
- [Top 1-2 items for tomorrow]
```

## EOW Loom Script

**When:** Every Friday before end of workday
**Where:** #ghostwriting-success-eod under EOW Thread

### Talking Points Structure (3-5 min)
1. **This Week's Wins** — student milestones, EEC launches, clients landed
2. **Challenges/Red Flags** — stuck students, payment issues, escalations
3. **Projections** — who's close to next milestone
4. **Next Week Focus** — top 3 priorities
5. **Team Callout** — anything helpful from team or clinics

## KPI Projections

**When:** Weekly, shared during standup
**Primary KPI:** Liftoff upsells

### Projection Criteria
For each student, assess:
- Assignment status (which module)
- Weeks in program
- Archetype (time commitment, drive level)
- Historical launch dates for similar students
- Recent activity velocity

### Format
```
Weekly Projections - [Date]

EEC Launches (next 2 weeks):
- [Student] - Assignment 3 submitted [date], targeting launch by [date]
- [Student] - On Module 2, archetype: full-time, likely launch by [date]

Paid Client Prospects:
- [Student] - Active outreach, X responses, sales call [pending/scheduled]

Liftoff Upsell Candidates:
- [Student] - $X clients landed, Week [X], ready for conversation
- [Student] - Approaching Week 16, needs Business Strategy Call

Student Referral Pipeline:
- [Any active referrals]
```

## Daily Standup Prep

**Topics to surface every morning:**
1. Contracts signed — who's outstanding, reach out
2. Failed payments — contact new, 2nd msg, 3rd msg, escalate
3. Projections — share top names for EEC launch and paid client
4. Daily theme — upsell (Tue) / call reviews (Wed) / curriculum (Thu)
